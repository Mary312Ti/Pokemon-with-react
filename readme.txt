Документация для компонентов React

App.js — это основная точка входа в приложение React. Он настраивает маршрутизацию для разных страниц с помощью react-router-dom и отображает соответствующие компоненты на основе URL-пути.

Зависимости
-react
-react-router-dom

Отрендеренные компоненты
-<PokemonList />: рендерит список покемонов, на URL-пути /.
-<PokemonDetails />: рендерит детали выбранного покемона, на URL-адресе /pokemon/:name.

Применение
Функция App возвращает элемент JSX, содержащий конфигурацию маршрутизации с использованием компонентов Routes и Route из react-router-dom.
Компонент <Routes> охватывает несколько компонентов <Route>, определяя пути и соответствующие компоненты для визуализации.
Свойство element используется для определения того, какой компонент отображать для каждого маршрута.


PokemonList.js

PokemonList.js — это компонент React, который извлекает и отображает список покемонов из PokeAPI. 
Он использует состояние и хук useEffect для управления извлечением и рендерингом данных.

Зависимости

-react-router-dom для компонента <link>
-axios для выполнения HTTP-запросов к PokeAPI

Применение
Функция PokemonList — это функциональный компонент, который извлекает список покемонов из PokeAPI при монтировании компонента.
Полученные данные сохраняются в состоянии pokemonList с помощью хука useState.
Хук useEffect используется для запуска выборки данных и обновления состояния pokemonList один раз при монтировании компонента.
Полученный список покемонов затем сопоставляется для создания списка элементов <li>, содержащих ссылки на страницу сведений о каждом покемоне.
Компонент <Link> из react-router-dom используется для создания интерактивных ссылок на страницу сведений о каждом покемоне, а имя параметра маршрута передается в URL-адресе.


PokemonDetails.js

PokemonDetails.js — это компонент React, который извлекает и отображает сведения о конкретном покемоне из PokeAPI. Он использует хук useParams для доступа к параметру маршрута 
и динамически извлекает соответствующие данные покемонов.

Зависимости
-react
-react-router-dom для хука useParams
-axios для выполнения HTTP-запросов к PokeAPI

Применение
Функция PokemonDetails — это функциональный компонент, который извлекает сведения о конкретном покемоне на основе параметра маршрута имени с помощью хука useParams из react-router-dom.
Полученные данные сохраняются в состоянии покемонов с помощью хука useState.
Хук useEffect используется для запуска выборки данных и обновления состояния покемона всякий раз, когда изменяется параметр маршрута имени.
Затем отображаются полученные данные о покемонах, включая их имя и фронтальное изображение с использованием свойства sprites.front_default.
